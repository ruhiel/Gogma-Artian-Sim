# Gogma-Artian-Sim

巨戟アーティア（Gogma Artian）のパーツ消費をシミュレーションし、効率的な厳選をサポートするツールです。

## 概要 (Overview)

このシミュレータは、特定の「当たり回数」に到達するまでに、どのタイプの遺装置をどれだけ消費すればよいかを自動計算します。

## 特徴 (Features)

- 自動最適化 厳選対象以外のパーツから、残り数が多いものを優先的に調整用として算出します。
- ステップ表示 当たりまでの調整工程と、当たりの瞬間をテーブル形式で一覧表示します。
- レスポンシブ シンプルなHTMLCSSJS構成で、ブラウザがあればどこでも動作します。

## 使い方 (Usage)

1. 各遺装置の現在所持数を入力します。
2. 厳選したいタイプ（攻撃会心属性）を選択します。
3. 調査済みの「当たり回数」をカンマ区切りで入力します。
4. 「実行」ボタンを押すと、消費手順がテーブルに表示されます。
